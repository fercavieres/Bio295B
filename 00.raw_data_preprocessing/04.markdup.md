# 04. Mark Duplicates

Se marcan los reads duplicados con `picard MarkDuplicates`. Los duplicados no se eliminan (`--REMOVE_DUPLICATES false`), solo se marcan. Las muestras con una sola corrida usan como input el `_sorted.bam`, las mergeadas usan el `_merged.bam`.

- **Output:** `*_dedup.bam` + `*_dedup_metrics.txt` por muestra
- **Herramienta:** `picard MarkDuplicates` (conda env: `picard`), `-Xmx20G`, `--TMP_DIR /data2/lab2`

---

## *S. parasiticus* — `04a.markdup_sparasiticus.sh`

```bash
#!/bin/bash
export TZ="America/Santiago"
source activate picard

bam="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/bam_completo_norte_final"

# Simples - input es _sorted.bam
files_simple="SPA1 SPA9 SRR22267306 SRR10019934 SRR10019945 SRR9946748"

for sample in $files_simple; do
    echo "[$(date '+%H:%M:%S')] MarkDuplicates: $sample"
    picard MarkDuplicates -Xmx20G \
        --INPUT $bam/${sample}_sorted.bam \
        --OUTPUT $bam/${sample}_dedup.bam \
        --METRICS_FILE $bam/${sample}_dedup_metrics.txt \
        --REMOVE_DUPLICATES false \
        --CREATE_INDEX true \
        --TMP_DIR /data2/lab2 && \
    echo "[$(date '+%H:%M:%S')] OK: $sample" || \
    echo "[$(date '+%H:%M:%S')] ERROR: $sample"
done

# Mergeadas - input es _merged.bam
files_merged="JJW5680 JJW4274 JJW4273 JJW730 T2-15 T7-11 HK_2019 UAMX5074 UAMX602 UAMX623 ZR_2019 AM_2019 AX_2016 JX_2019 T1-15 T14-19 T3-10 T4-10 TB1-15 TB1-16 TB2-16 TB3-11 TB4-11 TB6-11 TS17-16"

for sample in $files_merged; do
    echo "[$(date '+%H:%M:%S')] MarkDuplicates: $sample"
    picard MarkDuplicates -Xmx20G \
        --INPUT $bam/${sample}_merged.bam \
        --OUTPUT $bam/${sample}_dedup.bam \
        --METRICS_FILE $bam/${sample}_dedup_metrics.txt \
        --REMOVE_DUPLICATES false \
        --CREATE_INDEX true \
        --TMP_DIR /data2/lab2 && \
    echo "[$(date '+%H:%M:%S')] OK: $sample" || \
    echo "[$(date '+%H:%M:%S')] ERROR: $sample"
done

echo "============================================"
echo "MarkDuplicates S. parasiticus terminado"
echo "============================================"
```
