# 03. Merged Reads

Las muestras secuenciadas en múltiples corridas se fusionan en un único BAM por muestra con `picard MergeSamFiles`. Las muestras con una sola corrida (SPA1, SPA9, SRR*, SL5, SL8, SPO4, SPO7, SRR22267302, SRR22267303) no pasan por este paso.

- **Input:** `*_sorted.bam` por corrida
- **Output:** `*_merged.bam` por muestra
- **Herramienta:** `picard MergeSamFiles` (conda env: `picard`), `-Xmx20G`, `--TMP_DIR /data2/lab2`

---

## *S. parasiticus* — `03a.merge_sparasiticus.sh`

```bash
#!/bin/bash
export TZ="America/Santiago"
source activate picard

bam="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/bam_completo_norte_final"

# 3 corridas (pool2)
for SM in JJW5680 JJW4274 JJW4273 JJW730 T2-15 T7-11 HK_2019 UAMX5074 UAMX602 UAMX623 ZR_2019; do
    echo "[$(date '+%H:%M:%S')] Mergeando: $SM"
    picard MergeSamFiles -Xmx20G \
        --INPUT $bam/${SM}_high1_sorted.bam \
        --INPUT $bam/${SM}_high2_sorted.bam \
        --INPUT $bam/${SM}_low_sorted.bam \
        --OUTPUT $bam/${SM}_merged.bam \
        --TMP_DIR /data2/lab2 && \
    echo "[$(date '+%H:%M:%S')] OK: $SM" || \
    echo "[$(date '+%H:%M:%S')] ERROR: $SM"
done

# 3 corridas AM_2019 (pool3)
echo "[$(date '+%H:%M:%S')] Mergeando: AM_2019"
picard MergeSamFiles -Xmx20G \
    --INPUT $bam/AM_2019_high1_sorted.bam \
    --INPUT $bam/AM_2019_high2_sorted.bam \
    --INPUT $bam/AM_2019_low_sorted.bam \
    --OUTPUT $bam/AM_2019_merged.bam \
    --TMP_DIR /data2/lab2 && \
echo "[$(date '+%H:%M:%S')] OK: AM_2019" || \
echo "[$(date '+%H:%M:%S')] ERROR: AM_2019"

# 2 corridas (pool3)
for SM in AX_2016 JX_2019 T1-15 T14-19 T3-10 T4-10 TB1-15 TB1-16 TB2-16 TB3-11 TB4-11 TB6-11 TS17-16; do
    echo "[$(date '+%H:%M:%S')] Mergeando: $SM"
    picard MergeSamFiles -Xmx20G \
        --INPUT $bam/${SM}_high_sorted.bam \
        --INPUT $bam/${SM}_low_sorted.bam \
        --OUTPUT $bam/${SM}_merged.bam \
        --TMP_DIR /data2/lab2 && \
    echo "[$(date '+%H:%M:%S')] OK: $SM" || \
    echo "[$(date '+%H:%M:%S')] ERROR: $SM"
done

echo "============================================"
echo "Merge S. parasiticus terminado"
echo "============================================"
```
