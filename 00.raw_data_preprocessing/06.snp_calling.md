# 06. SNP Calling

El SNP calling se realiza en conjunto sobre todas las muestras de las cuatro especies. Antes de correr el script principal se generan dos archivos auxiliares: una lista con las rutas de todos los BAMs y una lista de los scaffolds a procesar.

- **Input:** `*_realign.bam` por muestra (output del paso de indel realignment)
- **Output:** `{scaffold}.vcf.gz` por scaffold en `/data2/lab2/lucila/BIO295B_FernandaCavieres/data/vcf_norte_final/`
- **Herramienta:** `bcftools mpileup + call` (conda env: `handsonVCF`), 40 procesos paralelos

---

### Paso 2 — Generar lista de scaffolds

```bash
for i in $(seq 1 20); do echo "scaffold_$i"; done > /data2/lab2/lucila/BIO295B_FernandaCavieres/data/chromosome_list.txt
```

### Paso 3 — SNP calling — `05.snp_calling.sh`

```bash
#!/bin/bash
export TZ="America/Santiago"
source activate handsonVCF

chromosome_list="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/chromosome_list.txt"
bam_list="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/bam_list_norte_final.txt"
ref="/data2/lab2/lucila/01.anotacion/SterChil1.hic.hap1.yahs_scaffolds_final.fa.MicroFinder.ordered.fa.masked"
output_dir="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/vcf_norte_final"

mkdir -p $output_dir

cat $chromosome_list | \
xargs -I {} -n 1 -P 40 sh -c \
'bcftools mpileup -Ou -f '"$ref"' -b '"$bam_list"' -d 5000 -q 10 -Q 30 -a "FORMAT/AD,FORMAT/DP,INFO/AD" --skip-indels -r {} --rf 2 | \
bcftools call -f GQ -vm -Oz -o '"$output_dir"'/{}.vcf.gz'

echo "============================================"
echo "SNP calling norte final terminado"
echo "============================================"
```
