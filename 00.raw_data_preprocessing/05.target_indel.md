# 05. Indel Realignment

Realineamiento local alrededor de indels usando GATK3. El proceso tiene dos pasos: primero `RealignerTargetCreator` identifica las regiones a realinear, luego `IndelRealigner` realiza el realineamiento.

- **Input:** `*_dedup.bam` por muestra
- **Output:** `*.realign.bam` por muestra
- **Herramienta:** GATK3 (conda env: `gatk3`), `-Xmx50G` para RTC, `-Xmx20G` para IR
- **Referencia:** versión con extensión `.fa` (requerida por GATK3)

> ⚠️ AM_2019 requirió un script separado con `--defaultBaseQualities 20` por problemas de codificación de calidades en el BAM. Se corrió después del script general de *S. parasiticus*.

---

## RealignerTargetCreator

### *S. parasiticus* — `05a.rtc_sparasiticus.sh`

```bash
#!/bin/bash
export TZ="America/Santiago"
source activate gatk3

ref="/data2/lab2/lucila/01.anotacion/SterChil1.hic.hap1.yahs_scaffolds_final.fa.MicroFinder.ordered.fa"
bam="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/bam_completo_norte_final"

files="SPA1 SPA9 SRR22267306 SRR10019934 SRR10019945 SRR9946748 JJW5680 JJW4274 JJW4273 JJW730 T2-15 T7-11 HK_2019 UAMX5074 UAMX602 UAMX623 ZR_2019 AM_2019 AX_2016 JX_2019 T1-15 T14-19 T3-10 T4-10 TB1-15 TB1-16 TB2-16 TB3-11 TB4-11 TB6-11 TS17-16"

for sample in $files; do
    echo "[$(date '+%H:%M:%S')] RealignerTargetCreator: $sample"
    gatk -nt 20 -T RealignerTargetCreator -Xmx50G \
        -R $ref \
        -I $bam/${sample}_dedup.bam \
        -o $bam/${sample}.realn.intervals \
        -allowPotentiallyMisencodedQuals -S LENIENT && \
    echo "[$(date '+%H:%M:%S')] OK: $sample" || \
    echo "[$(date '+%H:%M:%S')] ERROR: $sample"
done

echo "============================================"
echo "RealignerTargetCreator S. parasiticus terminado"
echo "============================================"
```

### AM_2019 (script separado) — `05e.rtc_ir_AM2019.sh`

```bash
#!/bin/bash
export TZ="America/Santiago"
source activate gatk3

ref="/data2/lab2/lucila/01.anotacion/SterChil1.hic.hap1.yahs_scaffolds_final.fa.MicroFinder.ordered.fa"
bam="/data2/lab2/lucila/BIO295B_FernandaCavieres/data/bam_completo_norte_final"

echo "[$(date '+%H:%M:%S')] RealignerTargetCreator: AM_2019"
gatk -nt 20 -T RealignerTargetCreator -Xmx50G \
    -R $ref \
    -I $bam/AM_2019_dedup.bam \
    -o $bam/AM_2019.realn.intervals \
    -allowPotentiallyMisencodedQuals -S LENIENT --defaultBaseQualities 20 && \
echo "[$(date '+%H:%M:%S')] OK RTC: AM_2019" || \
echo "[$(date '+%H:%M:%S')] ERROR RTC: AM_2019"

echo "[$(date '+%H:%M:%S')] IndelRealigner: AM_2019"
gatk -Xmx20G -T IndelRealigner \
    -R $ref \
    -I $bam/AM_2019_dedup.bam \
    -targetIntervals $bam/AM_2019.realn.intervals \
    -o $bam/AM_2019.realign.bam \
    -allowPotentiallyMisencodedQuals -S LENIENT --defaultBaseQualities 20 && \
echo "[$(date '+%H:%M:%S')] OK IndelRealigner: AM_2019" || \
echo "[$(date '+%H:%M:%S')] ERROR IndelRealigner: AM_2019"
```
