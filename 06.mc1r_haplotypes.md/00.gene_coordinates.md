# 00. Gene Coordinates — MC1R

Identificación de las coordenadas del gen MC1R en el genoma de referencia de *S. chilensis* a partir de la anotación funcional de BRAKER.

- **Genoma de referencia:** `SterChil1.hic.hap1.yahs_scaffolds_final.fa.MicroFinder.ordered.fa.masked`
- **Anotación funcional:** `/data2/lab2/lucila/01.anotacion/01.braker/02.filter_annot/MM_foh9j8_o.emapper.annotations.tsv`
- **GFF:** `/data2/lab2/lucila/01.anotacion/01.braker/02.filter_annot/SKchil_braker_sup100_longest.gff`

---

### Paso 1 — Buscar MC1R en la anotación funcional

```bash
cd /data2/lab2/lucila/01.anotacion/01.braker/02.filter_annot/

grep -i "mc1r" MM_foh9j8_o.emapper.annotations.tsv
```

Output relevante:
```
g3485.t1    9031.ENSGALP00000038021    1.69e-192    538.0    KOG3656@1
```

→ MC1R corresponde al gen **g3485.t1**

---

### Paso 2 — Obtener coordenadas en el GFF

```bash
grep "ID=g3485" SKchil_braker_sup100_longest.gff
```

Output relevante:
```
scaffold_12    AUGUSTUS    transcript    41127919    41128863    1    +    .    g3485.t1
```

→ MC1R está en **scaffold_12:41127919-41128863** (945 bp, exón único)

---

### Paso 3 — Extraer secuencia de ADN codificante

```bash
#!/bin/bash

ORIGEN="/data2/lab2/lucila/01.anotacion/01.braker/01.braker/braker.codingseq"
ID_GEN="g3485.t1"
SALIDA="mc1r_dna_skua.fasta"

awk -v id="$ID_GEN" 'BEGIN{RS=">";FS="\n"} $1 ~ id {print ">"$0}' "$ORIGEN" > "$SALIDA"

if [ -s "$SALIDA" ]; then
    echo "Archivo de ADN creado: $SALIDA"
    ls -lh "$SALIDA"
else
    echo "Error: No se encontró la secuencia."
fi
```

**Coordenadas finales MC1R:** `scaffold_12:41127919-41128863` — usadas en todos los pasos siguientes del pipeline.
